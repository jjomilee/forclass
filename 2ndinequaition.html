📌 꼭 기억할 핵심 정리 (a > 0)
D > 0 서로 다른 두 근 α, β 존재  (α < β)
f(x) > 0 x < α 또는 x > β
f(x) ≥ 0 x ≤ α 또는 x ≥ β
f(x) < 0 α < x < β
f(x) ≤ 0 α ≤ x ≤ β
D = 0 중근 α 하나 (x축에 접함)
f(x) > 0 x ≠ α인 모든 실수
f(x) ≥ 0 모든 실수
f(x) < 0 해 없음
f(x) ≤ 0 x = α
D < 0 실근 없음 (x축과 만나지 않음)
f(x) > 0 모든 실수
f(x) ≥ 0 모든 실수
f(x) < 0 해 없음
f(x) ≤ 0 해 없음
💡 핵심 원리: 이차함수 그래프가 x축 위에 있는 구간이 바로 f(x) > 0의 해예요.
판별식 D의 부호 → 그래프와 x축의 위치 관계 → 부등식의 해, 이 흐름을 항상 떠올려봐요!
